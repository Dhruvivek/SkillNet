
# 🎓 College Skill Network

A **college-exclusive networking platform** where students can discover peers based on skills, form hackathon teams, and ask technical or academic questions.

---

## 🚀 Overview

College Skill Network combines the power of:

* 🔗 Skill-based networking (like LinkedIn)
* ❓ Q&A system (like StackOverflow)
* 🤝 Hackathon team builder
* 🏫 College community platform

👉 The goal is to **help students collaborate, solve problems, and build projects together efficiently.**

---

## ❗ Problem Statement

Students in colleges often struggle to:

* Find peers with specific skills (Flutter, ML, Backend, etc.)
* Build hackathon teams quickly
* Get help with coding or assignments
* Discover talented students within their college

Current solutions like WhatsApp groups and random DMs are:

* Unstructured
* Inefficient
* Time-consuming

✅ This platform solves that by creating a **skill-based ecosystem inside the college**.

---

## 👥 Target Users

* College students
* Hackathon participants
* Developers building projects
* Students needing academic or coding help

---

## ✨ Features

### 👤 Skill-Based Profiles

* Name, branch, year
* Skills & interests
* Hackathon availability
* Actions: Connect, Invite, Message

---

### 🔍 Skill Discovery (Explore)

* Search students by skills (e.g., Flutter, Python)
* View profiles
* Connect or invite to hackathons

---

### 🧑‍💻 Hackathon Team Finder

* Find teammates based on required skills
* Invite students available for hackathons
* Instant chat on acceptance

---

### ❓ Q&A System

* Ask coding or academic questions
* Add tags (Flutter, DSA, ML, etc.)
* Answers, comments, upvotes

---

### 🔔 Skill-Based Notifications

* Get notified when relevant questions are posted
* Example: *"New Flutter question posted"*

---

### 📰 Community Feed *(Optional in MVP)*

* Post queries, ideas, or collaboration requests
* Like, comment, connect

---

## 🔄 User Flow

### 🟢 Onboarding

```
Login → Select Skills → Set Interests → Profile Created → Dashboard
```

---

### 🔍 Explore

```
Search Skill → View Profiles → Connect / Invite / Message
```

---

### 🤝 Hackathon Team

```
Select Skills → Find Students → Send Invite → Accept → Chat
```

---

### ❓ Ask Question

```
Post Question → Add Tags → Notify Relevant Users → Get Answers
```

---

### 🔗 Networking

```
Discover User → Send Request → Accept → Chat Enabled
```
